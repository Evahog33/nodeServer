
### 项目名称

- nodeServer

### 项目简介

- 一个node服务器的基本服务

### 目的

- 快速搭建一个简单的基础的node服务

### 环境要求

- nodejs（7.6+)
- mongodb（2.6+）

### 安装及开始

~~~
$ git clone https://github.com/Evahog33/nodeServer.git

$ cd nodeServer

$ npm install

$ npm install nodemon -g 

# serve with hot reload at localhost:9090
$ npm start
~~~

### 目录结构

- api 定义接口
- model 定义数据库
- public 提供静态文件
- app.js 项目入口文件
